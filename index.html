Curriculo

SQL

É a linguagem padrão para gerenciar bancos de dados relacionais, permitindo consultar, inserir, atualizar e excluir dados, além de modificar estruturas de banco.

Python

É uma linguagem de programação de alto nível, conhecida por sua sintaxe clara e concisa. Usada em áreas como desenvolvimento, automação e análise de dados.

Nuvem

Plataforma destinada à execução de aplicativos e serviços, baseada nos conceitos da computação em nuvem.

vs code

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft que inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.
